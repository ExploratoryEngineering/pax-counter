# mikroBUS PAX Counter

(Work in progress)

## BoM
* ESP-WROOM-32D
* PTS645 SMT switch
* 2 x 0402 10k resistor
* 1 x 6 pin 2.54mm pin header
* 2 x 8 pin 2.54mm pin header
